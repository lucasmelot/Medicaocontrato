# Landing Page - Brasilata

Landing page moderna, responsiva e profissional desenvolvida com **HTML5**, **CSS3** e **JavaScript puro**, criada para disponibilizar um vídeo tutorial e um canal direto de contato via Microsoft Teams.

---

## 📋 Sobre o Projeto

Este projeto foi desenvolvido seguindo boas práticas de desenvolvimento Front-end, com foco em:

- HTML semântico
- CSS organizado e reutilizável
- JavaScript moderno
- Responsividade
- Acessibilidade (WCAG)
- Performance
- Experiência do usuário (UX)

A identidade visual foi inspirada na marca **Brasilata**, utilizando tons de verde para transmitir profissionalismo, confiança e alinhamento visual com a empresa.

## ▶️ Como Executar

### Opção 1 - Execução Local

1. Baixe os arquivos do projeto.
2. Mantenha todos os arquivos na mesma pasta:
   - `index.html`
   - `style.css`
   - `script.js`
3. Abra o arquivo `index.html` em qualquer navegador moderno.

### Opção 2 - Hospedagem Web

O projeto pode ser publicado facilmente em:

- GitHub Pages
- Azure Static Web Apps
- IIS
- Apache
- Nginx
- Qualquer hospedagem HTML estática
